Called when user clicks on a context menu item.
contextMenuClick(recid, column, event)
recid string, id of the clicked record
column number, column index
event object, optional, DOM Event Object
Returns undefined.

Description

The first argument recid is the id of the record that was clicked, second is the column index. The third argument event is DOM event object. This method emits .onMenuClick event.
